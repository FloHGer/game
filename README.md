# The Game

---

## Instructions

###### Press "LEFT ARROW":
- turn left

###### Press "RIGHT ARROW":
- turn right

###### Press "UP ARROW":
- interact

###### Press "DOWN ARROW":
- open bag

#### with bag open:

###### Press "LEFT ARROW":
- previous item

###### Press "RIGHT ARROW":
- next item

###### Press "UP ARROW":
- select / deselect item

###### Press "DOWN ARROW":
- close bag
